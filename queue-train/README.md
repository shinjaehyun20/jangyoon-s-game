# 큐큐 기차역 (Queue Train)

자료구조 시각화 아이디어를 장윤이용 웹게임으로 번역한 미니게임.

## 핵심 구조
- **Queue 모델**: 기차 칸이 줄을 서고 맨 앞 칸부터 출발
- **정적 무대 / 동적 객체 분리**: 선로·역·대기칸은 고정, 기관차·기차칸·파티클은 상태에 따라 이동
- **Model / View / App 분리**: `QueueModel`, `View`, `App` 역할로 나눔

## 플레이 방법
1. 시작 버튼을 누른다.
2. 맨 앞 칸의 색과 그림을 본다.
3. 같은 색 역 버튼을 누른다.
4. 맞히면 점수와 콤보가 오르고, 틀리면 하트가 줄어든다.

## 파일 구성
- `index.html` : 구조 및 UI 마크업
- `style.css` : 선로/역/기차칸 시각 스타일
- `script.js` : Queue 모델 + 렌더링 + 게임 제어
- `thumb.svg` : 목록 썸네일
